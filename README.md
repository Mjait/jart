# jart
The purpose of this exercise is to create a PNG image by drawing simple geometrical shapes into a pixel buffer.

## Project structure
- `geometrical_shapes/`: Java sources (shapes + rendering)
- `image.png`: output image generated by `Main` (created when you run the program)

## How to run
From the repo root:
```bash
javac geometrical_shapes/*.java
java -cp geometrical_shapes Main
```

This generates `image.png` in the current working directory (usually the repo root).
